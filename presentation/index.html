Sass: Nested rules & properties
Sass allows CSS rules to be nested within one another. The inner rule then only applies within the outer rule’s selector. CSS has quite a few properties that are in “namespaces;” for instance, font-family, font-size, and font-weight are all in the font namespace. In CSS, if you want to set a bunch of properties in the same namespace, you have to type it out each time. Sass provides a shortcut for this: just write the namespace once, then nest each of the sub-properties within it.
